# Quiz App

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript.
This project displays multiple-choice questions, tracks score, and shows final results.

## Features

* Start quiz screen
* Multiple-choice questions
* Score tracking system
* Progress bar indicator
* Instant answer feedback (correct / incorrect)
* Final result message based on performance
* Restart quiz option
## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

##  How It Works

1. Click Start Quiz
2. Answer each question
3. See immediate feedback (correct/incorrect)
4. View final score at the end
5. Restart quiz anytime

##  Quiz Logic

* Questions are stored in a JavaScript array
* DOM elements are dynamically generated
* Score increases when correct answer is selected
* Progress bar updates with each question

This project helped me improve:

* DOM manipulation
* Event handling in JavaScript
* Working with arrays and objects
* Dynamic UI updates
* Basic UX logic for interactive apps
